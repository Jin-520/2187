# 2187 Game

一个基于网页的 2048 风格益智游戏，目标是将数字方块合并达到 **2187**。

## 🎮 游戏介绍

这是一个经典的滑动方块益智游戏。通过上下左右滑动，将相同数字的方块合并，最终目标是合成出 **2187** 这个数字方块。

## ✨ 特性

- 🎯 **经典玩法**：简单易上手，难以精通
- 🎨 **精美界面**：简洁优雅的 UI 设计
- 📱 **响应式设计**：支持桌面和移动设备
- 🖱️ **多种操作方式**：
  - 键盘方向键（↑ ↓ ← →）
  - 触摸滑动（移动设备）
- 🏆 **分数系统**：实时追踪你的得分
- 🔄 **重新开始**：随时开始新游戏
- ✨ **动画效果**：流畅的方块移动和合并动画

## 🎯 游戏规则

1. 游戏在一个 4×4 的网格上进行
2. 每次移动后，会在空白位置随机生成一个新方块（2 或 4）
3. 使用方向键或滑动手势移动所有方块
4. 当两个相同数字的方块碰撞时，它们会合并成一个新方块，数字相加
5. 当无法再进行任何移动时，游戏结束
6. 目标是合成出 **2187** 方块！

## 💡 游戏技巧

- 🧠 **提前规划**：尽量保持最大数字的方块在角落
- 📍 **保持稳定**：避免随意打乱方块的排列顺序
- 🔢 **优先合并**：优先合并较小的数字方块
- ⚠️ **谨慎使用上移**：避免将大数字方块分散
